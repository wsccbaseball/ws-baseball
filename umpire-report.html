Metric Definitions
Zone Tendency — compares ball accuracy to strike accuracy. Squeezer: misses called strikes more — pitchers should work the zone edges. Expander: calls strikes outside the zone — hitters beware of chasing. Neutral: within 3 points either way.
Consistency — highest minus lowest single-game accuracy. A small range means a predictable ump; a wide range means performance varies game to game.
Strike vs Ball Gap — ball accuracy minus strike accuracy as a signed number. Positive = better at calling balls (squeezer). Negative = better at calling strikes (expander).
Avg Favor Runs — average RE24-based run value of favoritism per game. Higher values mean miscalls had a larger run impact on average.
